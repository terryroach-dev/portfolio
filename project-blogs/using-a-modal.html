Using a Modal

A modal is a dialog box/pop up window that is displayed on top of the current page

04/03/2021

A modal is a dialog box/pop up window that is displayed on top of the current page. The user must engage with the modal in order to close and return to the main content. Modals can be used to direct a users attention to some required action or some important information. You will often see them used at the bottom of pages prompting the user to accpet cookies.

In this example I set up a full page modal which when opened dislpays a little bit of information about me. To create the modal required HTML to set the elements structure and add the content required, CSS to add the styling and Javascript to give the modal it's functionality.

1. Set up the 3 files to be used, index.html, styles.css and modal.js.

2. Set up the basic html elements such as the html, head, title and body tags. Add the css file and javascript links to the html file.

3. Add the html element Skeleton. This is a div with an id of 'overlay', inside of that is another div with an id of 'modal'. These 2 divs will be where the modal is placed. Another div is added with a class of 'main-content', this will be where the main content is the page will be placed.

4. Complete the html file by adding the content which includes a twitter icon from Font Awesome and 2 buttons which will be used to open and close the modal

5. Added some of the styling to CSS file, focussing on mobile first. The styling included general body properties such as margin and padding, font properties, links and the buttons. I went back to the html file and added a link to my twitter acc and google fonts.

6. It is time to hook up the buttons in Javascript set the functionality. The javascript will grab the 2 buttons and the 'overlay' div. The 2 buttons are given an event listener which when clicked will either open or close the modal by setting the 'display' property of the 'overlay' div to 'block' or 'none'.

7. The final styling is added, this includes setting the 'overlay' div to 'display: none' so the modal is not viewable when the page first loads. It also includes the media queries for bigger screens.
